Apply Angelov GaN Model Extracted by IC-CAP in ADS

by Feng Gu [Keysight] May 22, 2017

This blog explains how to apply the Angelov GaN model extracted by IC-CAP in ADS(Keysight Advanced Design System), for large signal simulation verification and/or other circuitry simulation. The difficulty was caused by the fact that Angelov GaN model is in VerilogA format and is somehow different from ADS’s built-in models.

Here are the steps for ADS users to use ICCAP-saved .mps and .lib files.

1. Add the VerilogA designKit into ADS in order to handle the verilogA code of Angelov GaN model.

(1) In ADS main window, do “DesignKit > Manage Favorite Design Kits”, open the “Manage Favorite Design Kits” dialog.
(2) Click on “Add Library Definition File”, and select the $HPEESOF_DIR/tiburonda/ads/designkits/tiburon-da_veriloga directory.
(3) Pick up the “Lib.defs” file, and click Open.
(4) From the “Add Design Kit” window, add the it to the current workspace, click OK.
(5) Now, close the Manage Favorite Design Kits window, and open a Schematic window, we now see “Devices-Verilog-A” available in the Component Palette. As shown below.



2. Select one of the icons highlighted in the red rectangle for Angelov model.

3. Load the model card, and there are two ways to do it:
(1) do “Tools > IC-CAP Import > Any Device”, as shown below.


Input “angelov_gan_va_Model” (Case sensitive).


A Dialog will pop up, go ahead to load the ICCAP-saved .mps file.
We shall see the model parameters are loaded in the Schematic.

Note: this method would allow user to further adjust model parameters.

(2) We could also use “DynamicLink > Add Netlist File Include”



Then, pick the .lib file saved from ICCAP. Note, the path of the VerilogA file in the lib file must be correct.


Note: this method won’t allow user to further adjust model parameters.
