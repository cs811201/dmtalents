How to Change 2-port Network Input and Output Ports

MQA FAQ

By default, for a transistor, MQA uses Gate as the Input port and Darin as the Output port for 2-port network application, to simulate and calculate S and Y parameters.

In case we want to change the Input and Output ports assignment to a different device terminal, here is the way:

The In/Out ports are specified in the "model.icf" file, and each MQA project requires a "model.icf" file, so this arrangement will be project dependent.
Of course one could also change the master copy of the "model.icf" file to make the changes global.

Take BSIM4 model as an example, the default "model.icf" file is located at:
    $MQAHOME/etc/model/$simulator/mos/bsim4(54)/model.icf


In the [PORT] section, the highlighted 3rd item in the curly braces defines the In/Out or GROUND ports.
  • 1 stands for Input port
  • 2 stands for Output port
  • 0 stands for GROUND
So, we can change the numbers here to adjust the Input and Output ports for 2-port network simulation.

S/Y/Z/H/T parameters can be later on converted to one another when needed.
