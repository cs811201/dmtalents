Link ICCAP to PSpice Simulator

ICCAP FAQ

This note explains how to link ICCAP to PSpice Simulator.

Note: It's recommended to read the Link ICCAP to External Simulators in General firstly, including troubleshooting tips.

Use the Download Link for OrCAD PCB Designer Lite, which is the free version.

In $ICCAP_ROOT/iccap/lib/usersimulators look for line for pspice alias, and edit to look like the following.

# pspice  pspice   $CDSROOT\tools\pspice\psp_cmd.exe   ""   CANNOT_PIPE
pspice pspice C:\Cadence\SPB_17.2\tools\bin\psp_cmd.exe   ""   CANNOT_PIPE

Then restart ICCAP.

Verification of simulation link to PSpice

  1. load the following *.mdl file:
    â€¦/Examples/model_files/mosfet/nmos3.mdl

  2. In the model Variables, add SIMULATOR = pspice

  3. Go to the /large/idvg/ setup, do Clear -> Simulated so that any previously saved simulatioin will be gone.

  4. Simulate.

  5. Open the /large/idvg/idvsvg plot, confirm that the simulated data appears.

    Note: the simulation data are solid lines on the plot.
