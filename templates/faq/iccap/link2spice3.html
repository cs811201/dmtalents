Link ICCAP to Spice2, Spice3, HPSpice

ICCAP FAQ

Good news! Spice2, Spice3 and HPSpice are built-in, we can use them directly in ICCAP, and no need to define any entry in the $ICCAP_ROOT/iccap/lib/usersimulators config file.

Simply declare the SIMULATOR varible to one of them, it will start to work. As shown below:


Verification of simulation link to Spice3

  1. load the following *.mdl file:
    …/Examples/model_files/mosfet/nmos3.mdl

  2. Make sure spice3 is assigned to SIMULATOR.

  3. Go to the /large/idvg/ setup, open the ‘idvsvg’ plot and clear the simulated data

  4. Run a simulation and confirm that the simulated data appears on the ‘idvsvg’ plot.



Note: we can change spice3 to spice2 or hpspice in this example to verify them.
