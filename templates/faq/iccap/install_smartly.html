Install IC-CAP Smartly on Windows

ICCAP FAQ


by Franz Sischka (www.SisConsult.de), and Reint Lange
Note: on Windows, only 64-bit Operating Systems supported!

After IC-CAP gets successfully installed and licensed, we can see the following IC-CAP Main windows.



Step 1. Preparation

First of all, create a good Directory Structure
  • for your code downloads,
  • licenses,
  • IC-CAP and
  • ADS
  • your own data files


Step 2. Setup License Server
(Skip if you have IT admin to take care of the license server setting for you.)
Download the License server from Keysight official website to your PC ...
  • www.keysight.com/find/eesof-iccap-latest-downloads






    • and install it into directory
  • C:\A_ICCAP\_Licenses


    • When done, save your license file into this directory too.
    Step 3, download IC-CAP installer

    Come back to the following website:
  • www.keysight.com/find/eesof-iccap-latest-downloads




    • Make sure the Windows radio button is selected, and then click the red Download button to start the downloading process. And install it into directory:
  • C:\A_ICCAP\ICCAP_2020



    • Best Practice: name your IC-CAP executables(Install) directory after year and version, e.g. ICCAP_2020, or ICCAP_2020_Update2 when it’s a version with updates.

    Next, name your IC-CAP user directory after year and version, as shown below.


    Next, point the License Tools directory to which we had already installed the licenses in Step 2.


    Verify all, before clicking 'Install'.


    Now we have ICCAP successfully installed.

    Step 4. Download and install Keysight ADS

    The purpose is to use ADS as the simulator in ICCAP, please refer to other application notes if you wish to use other simulators(such as Hspice or Spectre, etc.) for model simulation in ICCAP.

    In the same way, use the following link to download ADS from the Keysight website, and install it as follows:
  • www.keysight.com/find/eesof-ads-latest-downloads



    • Step 5. Link to ADS simulator

    Enable the link from IC-CAP to ADS, by setting the Windows Environmental Variable 'ADS_DIR':


    Step 6. Start IC-CAP

    Besides ICCAP’s main window, while ICCAP is starting itself we shall see the Status window reporting “… license granted.”. Then IC-CAP is ready to work, as shown below.

    Step 8. Test the link between IC-CAP to ADS.
    For more info, please refer to this application note.
    About ADS-under-ICCAP Licensing
    When called from IC-CAP, the hpeesofsim.exe (ADS) gets a license from IC-CAP for simulations of:
      > DC
      > CV
      > Spar
      > 1/f Noise
      > Harmonic Balance(HB)
      > Verilog-A
    This license also enables the use of ADS in Spectre and HSpice netlist syntax !


    Once open the "1_diode_demo_ADS.mdl" model window, follow the following 5 steps to run ADS simulation marked in the next two screenshots.
    When we get newly generated simulation curve on the IV plot, then the linkage is successfully established.




    Step 9. Linking to Instruments

    Get an 82357B USB/GPIB interface High-Speed USB 2.0 cable, as shown below.
  • https://www.keysight.com/en/pd-851808-pn-82357B/usb-gpib-interface-high-speed-usb-20?nid=-32516.426029.00&cc=GB&lc=eng



    • And, download and install Keysight IO Libraries Suite
  • https://www.keysight.com/en/pd-1985909/io-libraries-suite?nid=-33330.977662.00&cc=GB&lc=eng



    • install it into the default directory presented during the installation process.
    During installation, answer all questions with the default answer.
    When the driver software is installed, plug-in your USB-GPIB interface.
    • It will first display the Fail LED,
    • then all LEDs,
    • and finally the Green Ready LED only.



    Activate the GPIB Interface in IC-CAP
    In IC-CAP's Hardware Window, add the interface as gpib0 (all small characters)


    Step 10. Last but not least
    Getting Started with IC-CAP



    Where to find more examples:


    For expert, if you want to auto-load a ModelFile when starting IC-CAP.


    If you want to IC-CAP and auto load a ModelFile and also to auto-execute a Macro within this ModelFile (which could then activate other features), enter into field "Target' above the following: 
    C:\A_ICCAP\ICCAP_2020/bin/iccap.exe \Path\ModelFileName -x /modelname/macroname
    Note: the above “-x macro” is used to specify the Macro, and it's optional.
