Change default mb_set directory location

MBP FAQ

When MBP starts, it will create a few run-time configurations or setting files under $HOME/mb_set directory.
For example, mine is C:\Users\shuancai\mb_set, as shown below.



This directory will be automatically created the first time we start MBP, if it doesn't exist.

Note: on Linux, it will be a hidden directory, namely $HOME/.mb_set.

And, there is one option called MBSETPATH that allows us to change this directory to a user specified location. This option needs to be added to the $MBPHOME/mbp.bat wrapper file. As highlighted below.


For example, here I set it to “C:\Users\shuancai\tmp”, an arbitrary path user specified.
To pass in a variable to the Java program, we need the “-D” keyword to assign it, as follows:
    -DMBSETPATH="C:\Users\shuancai\tmp"

Once this change is saved, re-start MBP, we will start to see all the run-time files get saved under C:\Users\shuancai\tmp.

This option should work for both Windows and Linux.
