Alternative Optimization Error Function in Script

We've learnt that we can do Optimization in MBP Script, where this error function is used: 

double errorFunc(){
	return cmd.getRMS();
}
However, sometimes one may find this function not working, RMS won't further reduce. This is a known issue and here is the workaround.

Please try using the following alternative Error Function, as shown below:


Here is the code for copy-paste: 

import com.accelicon.modelbuilder.optimization.optimizer.conf.OptimizerConfig;
import com.accelicon.modelbuilder.ModelBuilder;

// your other code

double errorFunc(){
            OptimizerConfig opt = OptimizerConfig.getInstance(ModelBuilder.framework);
            return opt.getGUIRMS(false); // false means not to use weight setting.
}
