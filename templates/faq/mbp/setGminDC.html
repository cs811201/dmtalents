Set GminDC

GMINDC, as an option in many SPICE simulators, helps overcome DC convergence problems during simulation.

By default, GminDC = 1E-12.

To change GminDC to another value, in MBP, can be done in the following:

Please go to MBP menu "Extraction -> Options -> SPICE Options", we could actually set a few options here:
  • GminDC
  • Scale
  • Geoshrink
  • CV Freq
