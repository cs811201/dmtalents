Optimizer Options

MBP provides more than 10 built-in Optimizers for Optimization, as shown below:



Below are some explanation to these options when we come to take a closer look at them.

Use MBP menu "Optimization -> Choose Optimizer" to open the following window:



By unchecking these checkboxes to the right of each option, we will be able to change them.
  • VTR: Value to reach. The optimization will stop its minimization if either the maximum number of iterations is reached or the best parameter vector found makes the objective function value less than VTR.
  • IterMax: Maximum number of iterations.
  • TOL: Tolerance of the parameter optimized.
  • SynchStep: The optimizer will exert convergence check every “SynchStep” iterations.
  • SynchMax: Maximum number of iterations for tolerance convergence.
