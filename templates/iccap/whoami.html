WHOAMI

ICCAP Application Note

It is very useful for us to know a Macro or Transform’s path or name at run-time, so that we can do some dynamical programming, also very good for logging purposes or troubleshooting.

ICCAP PEL comes with a few query function, let’s take a look at WHOAMI().

As the function’s name implies, it will return current function’s full name. For example:


We can use it with or without any arguments.

1. when no arguments are used, it returns current Macro or Transform’s full path, as shown above.

It’s a good practice to insert such a print statement at the beginning of each Transform or Macro, so that we could see ICCAP runs what at run-time in the Status window, a technical to monitor our code.

2. it supports the following 2 argument keywords, as shown below.
    (1) Type, reports it is a Macro or a Transform.
    (2) PartName, returns only the name instead of the full path.



Here is the code for copy-paste: 
print "running: " & whoami()

print "Type: " & whoami("Type")

print "Just name: " & whoami("PartName")
