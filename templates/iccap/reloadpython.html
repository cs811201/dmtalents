Dynamically Reload External Python Modules

ICCAP Application Note

The first time ICCAP loads external python files(.py), those .py files got compiled to .pyc files(binary), as shown below.



And, any changes made to those external .py files after their .pyc files were created wonâ€™t refresh(re-compile) the .pyc files, unless these .pyc files were deleted.

Luckily, Python comes with a built-in function called reload() which can help us dynamically reload our external python modules, as shown below, line 8 and 9 reload these two Python modules so that our Python code in ICCAP will always reflect the latest external code.



Here is the code for copy-paste: 
import sys, os
WorkingDir = "C:\work\ICCap\USER_PYTHON"
sys.path.append(WorkingDir)

import iccap_util as p_until
import iccap_vth as p_vth

p_until= reload(p_until)
p_vth = reload(p_vth)
