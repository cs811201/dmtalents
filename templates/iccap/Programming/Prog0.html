COPY-PASTE PROGRAMMING EXAMPLES

Foreword

by Franz Sischka, www.SisConsult.de
IC-CAP is an open software which enables the user to implement his own device modeling ideas with basically no restrictions.

This is due to the fundamental concept of IC-CAP, that, from the very beginning with version 1.0,
  • Complex numbers are supported required to handle impedance simulations, S-parameters etc.
  • As well as data sets of complex numbers S-Parameters as function of frequency and bias.
  • Matrix operations are supported e.g. inversion of ABC-Matrices for de-embedding.
  • And, the handling of data sets is very simple e.g. gm = DERIVATIVE(vd, id, 1)

However, writing the corresponding program code is not always intuitive.

Therefore, to make the user's life easier, this chapter of the IC-CAP Blog features a collection of the most important copy-paste examples .
