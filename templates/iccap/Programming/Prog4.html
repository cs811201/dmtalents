Finding And Identifying Objects

by Franz Sischka, www.SisConsult.de
When programing IC-CAP activities, the dynamic identification of current child and parent objects is important. This is provided by the Program Keywords 'ICCAP_FIND_CHILDREN', 'WHOIS' and 'WHOAMI'.

Note: see a Video demo at the end of this page.

Identifying Child Objects

ICCAP_FIND_CHILDREN ".", "Plots", PlotNames

automatically turns the variable (e.g. 'PlotNames') into an ICCAP_ARRAY holding the names of the searched items "Plots", located in the Setup "." of the current Transform).

Search Item Keywords are:
    Model
    Macro
    DUT
    Setup
    Input
    Output
    Transform
    Plot
    GUI Item

Examples:
Set Variable 'myModelNames' to a list of all currently loaded Model Projects:
    ICCAP_FIND_CHILDREN "/", "Model", myModelNames

Set Variable 'myPlotNames' to a list of all Plots present in the Setup of the current Program Transform:
    ICCAP_FIND_CHILDREN ".", "Plot", myPlotNames

Identifying Parent Objects

ParentSetupName = WHOIS(".", "Parent Setup")
returns the name of the Setup in which the command was executed, into Variable ParentSetupName.

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

Identifying The Current Transform Name

TransformName = WHOAMI(".", "PartName")
returns the name of the Transform in which the command was executed, into Variable TransformName.
