This ModelFile hosts a simple example of how to program a library inspection feature in IC-CAP.

After loading and opening the example file in IC-CAP, have first a look to Tab 'Macros'.


The link to the library is defined in Tab 'Circuit':


The location and name of the NMOS library example is linked by the line #echo include ...

To link to the user-selected library cell, we make use of the IC-CAP Netlist Parser Control feature $VAR(), which enables to link the contents of a user-defined Variable, here 'Library_CellName', directly into the netlist.

Note: like before, the line starts with the IC-CAP keyword #echo, which switches-off the automatic netlist parsing of IC-CAP for this particular line (because we apply in this line non-standard parser features).

The 'Library_CellName' is set by a user interface (GUI), defined in Setup 'Inspect_Library', and Transform 'SELECT_LIBRARY_CELL', as depicted below:
