Display Target on Y2 Axis

On IMV Plots, Axis[Y2] is the Y axis on the right, for example, we have Ids_Vgs plot, and overlay Gm_Vgs and put Gm on the Y2 Axis to the right, as shown below:



Recall that we have this Axis[Y2] option on XY Graph when creating an IMV Graph, as shown below:

Where we put Ids on Axis[y], and Gm on Axis[y2]. It is this simple except that we have to include both Gm and Ids in the same IMV Target Data , otherwise in the Axis[y2] field, we cannot point to a second target.

Let's assume we begin with the existing Gm IMV target, and this example can be found as ScriptZipFiles/08_IMV_Y2.zip

Because Ids is not one of the default IMV targets, we have to create one firstly. As shown below.
let's do a New -> Data, and just need to select("Ids_Vgs_Vbs") and return it.


Now switch back to Gm, right click on it and New -> Graph to create an XY Graph, name it "Ids_Gm_Vgs". And Attached Data: Ids brings Ids Data to current Gm Data, so that we can view both of them on one plot. As shown below.
