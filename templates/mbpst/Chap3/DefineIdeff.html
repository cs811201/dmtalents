Define Ideff

Ideff is defined in the following fashion, and it's a perfect case to use Expression type of IMV definition. From Ids_Vds_Vgs plot, we have:


Ideff = ( Idhigh + Idlow ) / 2
Idhigh is Ids @ Vds=Vdd/2, Vgs=Vgg
Idlow is Ids @ Vds=Vdd, Vgs=Vgg/2.
