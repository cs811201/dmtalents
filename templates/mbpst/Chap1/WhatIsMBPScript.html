What is MBP Script

MBP Script is a Java™ based scripting language, object oriented, no compilation is required at user end, just code as we go. And, many MBP specific APIs(Application Program Interface) and high efficiency programming structures are nicely introduced for Device Modeling work. MBP Script runs equivalently on Windows and Linux.

On the other hand, MBP Script window is also full of GUI based user-friendly interfaces which allow us to view/create many things without any coding, MBP knows how to translate GUI input into scripts under the hood.

MBP Script can be used to access, define, customize, and automate a lot of MBP objects and operations, including:
  • Model Parameters
  • Measurement Data
  • IMV(Intermediate Variables), many are pre-defined, such as Vth, Gm, Gds, Idlin, Idsast, etc.
  • Scaling Plots, such as Vth vs. L, Idsat/W vs. W, etc.
  • Math Transformation, such as Line Fitting, Data Analysis, etc.
  • Simulation and Optimization
  • Extraction Flow Automation
  • GUI Components
  • And all kinds of things that Java™ can do, such as: File Management, Invoke System Executables, etc...

  • Plus, MBP Script comes with the Inline Debugging capability! (see Chap4.2)

For Example

To get a Model Parameter value, say, currently we have a BSIM4 model loaded in MBP, and we wanted to check the value of "Vth0" from the script, as follows:


Here, Param and MBP are built-in objects ready for user to access things from MBP environment, they don't require additional definition beforehand, just use them in Script whenever we want.

Example Script code is shown below: 
//Get Model Parameter "Vth0" value.
Param pVth0 = MBP.getParam("Vth0");
double v = pVth0.getValue();

System.out.println("Vth0=" + v);
