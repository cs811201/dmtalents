PageGroup APIs

PageGroup build(String expression)

Description: Organizes the page group to a tree. The expression is the same as in DATA.build(â€¦)
Example 
def group = cmd.getPageGroup("ids_vgs_vbs");
group = group.build("select(w == min(w) && l==min(l))");
cmd.plot(group.toPagePlots());

PageGroup get(int index)

Description: Gets the sub page group at index.
Example 
def group = cmd.getPageGroup("ids_vgs_vbs");
group = group.build("splitBy(w)");
cmd.plot(group.get(0).toPagePlots());

int size()

Description: Gets the number of sub groups.
Example 
def group = cmd.getPageGroup("ids_vgs_vbs");
group = group.build("splitBy(w)");
System.out.println(group.size());

PageGroup filterBy(DATA data)

Description: Filters the plots with data.
Example 
def group = cmd.getPageGroup("ids_vgs_vbs");
DATA d = MBPDATA.get("[device]");
d = d.build("select(w == max(w) && l == max(l))");
group = group.filterBy(data);
cmd.plot(group.toPagePlots());

PageGroup filterBy(DATA data, String restrictions)

Description: Filters the plots with data.
Example 
def group = cmd.getPageGroup("ids_vgs_vbs");
DATA d = MBPDATA.get("[device]");
d = d.build("select(w == max(w) && l == max(l))");
group = group.filterBy(data, "w,l");
cmd.plot(group.toPagePlots());

PagePlot[] toPagePlots()

Description: Returns all the plots.
Example 
def group = cmd.getPageGroup("ids_vgs_vbs");
cmd.plot(group.toPagePlots());
