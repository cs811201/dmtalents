MBPOPT APIs

MBPOPT provides some methods to do optimization.
MBPOPT is an internal variable but not a class.

void setOption(String name, String value)

Description: Sets Optimizer options, including optimizer type and others.

String getOption(String name)

Description: Gets option of Optimizer.

boolean setError(sfunc error)

Description: "error" is the error function which returns double. Optimizer will try to minimize its value to zero.

boolean setCheck(sfunc check)

Description: "check" is a function which returns Boolean. Optimizer will exclude the solutions that return value is false. If it is set to null, the optimizer will ignore the check.

double[][] optimize(Param[] params)

Description: Optimizes with "params". The return values are the best results that optimizer finds. One may use MBPOPT.POPULATE_PARAM to save it for next optimization run to obtain a better result.

double[][] optimize(Param[] params, double[][] initial)

Description: Description Optimizes with "params". "initial" are several former results, which the optimization start from. The user may get them from MBPOPT.POPULATE_PARAM. The return values are several best results that optimizer found. The user may use MBPOPT.POPULATE_PARAM to save it for next optimization, and may get a better result of next optimization.
