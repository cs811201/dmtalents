CMD APIs

CMD provides a lot of APIs to deal with Plot, Region, Data File, Model Parameter, Optimization, GUI components, File, etc.

PagePlot[] plot(String plotExpression)

Description: Plots the graphs with “plotExpression”, and returns the plots.

CMD.plot("analysisname@condition1,condition2...@plotInfo option1, option2...");
CMD.plot("navigatorname.graphname@ condition1,condition2...@plotInfo option1, option2…");

Options can be:
xScale = log/linear
yScale = log/linear
region=[#vgg/2,#vgg]
range=[x_lower,x_upper,y_lower,y_upper]
math=mathName
Example 
CMD.plot("ids_vgs_vbs");
CMD.plot("imv.imv.vth.vth_w_l");
CMD.plot("ids_vgs_vbs@w=1u,l=1u,abs(vds)<0.5");
CMD.plot("ids_vgs_vbs@plotInfo xScale=log,math=y'");

void plot(PagePlot[] plots)

Description: Plots the graphs.

PageGroup[] getPageGroup(String graphName)

Description: Gets a page group with “graphName”, and one can use the page group to select a graph to plot.
CMD.getPageGroup("analysisname");
CMD.getPageGroup("navigatorname.graphname);
Example 
CMD.plot("ids_vgs_vbs");
CMD.plot("imv.imv.vth.vth_w_l");

void clearPlots()

Description: Removes all the plots from the browser. And it will also clear the selected regions.
Example 
CMD.clearPlots();

void clearParam()

Description: Removes all the selected parameters.
Example 
CMD.clearParam();

void clearRegion()

Description: Removes all the selected regions.
Example 
CMD.clearRegion();

void selectParam(String name)

Description: Selects parameters by name.
Example 
CMD.selectParam("vth0");

void setParamValue(String name, double value)

Description: Sets parameter value.
Example 

//if there are several bins, and each bin has "vth0", only the "vth0" in current bin will be changed.
CMD.setParamValue("vth0",0.3);

//”nmos” is the bin name
CMD.setParamValue("nmos.vth0",0.3);

double getParamValue(String name)

Description: Returns the parameter value.
Example 
//if there are several bins, and each bin has "vth0", only the "vth0" in current bin will be changed.
double v = CMD.getParamValue("vth0");
System.out.println(v);

//”nmos” is the bin name
double v = CMD.getParamValue("nmos.vth0");
System.out.println(v);

boolean setParamBoundary(String name, double lower, double upper)

Description: Set soft boundary of parameter
Example 
CMD.setParamBoundary("vth0",0.1,0.7);

CMD.setParamBoundary("nmos.vth0",0.1,0.7);

boolean setParamHardBoundary(String name, double lower, double upper)

Description: Sets hard boundary of parameter
Example 
CMD.setParamHardBoundary("vth0",0.1,0.7);

CMD.setParamHardBoundary("nmos.vth0",0.1,0.7);

void optimize()

Description: Performs optimization with the selected plots and parameters.
Example 

CMD.optimize();

void addMenu(String name, sfunc action)

Description: Adds a menu to MBP.
The user can define what to do in "action".
Example 
CMD.addMenu("customize->clearData",action);
void action(){
	cmd.clearData();
}

void clearMenu()

Description: Removes all the menu defined in MBP scripts.
Example 
CMD.clearMenu();

void loadTasktree(String name)

Description: Loads the task tree by name.
Example 
CMD.loadTasktree("autoBin.bsim3");

void runTasktree()

Description: Runs task tree.
Example 
CMD.runTasktree();

void loadModel(String file)

Description: Loads a model from a file.
Example 
CMD.loadModel("C:\\nmos.l");

void saveModel(String file)

Description: Saves current model to a file.
Example 
CMD.saveModel("C:\\nmos.l");

void loadData(String file)

Description: Loads data from a file.
Example 
CMD.loadData("C:\\nmos.mdm");

void clearData()

Description: Removes all the data in MBP.
Example 
CMD.clearData();

void exec(String command)

Description: Runs a system command.
Example 
CMD.exec("notepad");

String readFile(String file)

Description: Reads a file and returns its content.
Example 
String content = CMD.readFile("c:\\a.txt");
System.out.println(content);

void saveFile(String file, String content)

Description: Saves “content” to a file.
Example 
CMD.saveFile("c:\\a.txt","sample");

void repaint()

Description: Refreshes all graphs.
Example 
CMD.repaint();

double getRMS()

Description: Gets the RMS error value in the selected graph.
Example 
double rms = CMD.getRMS();
