String[] getBinPointNames(String expression)

Description: Gets the bin names with expression.
For example:
Model model = …;
model.getBinNames("w>1u && l < 1u");
It will return all the bins which satisfy “w>1u && l < 1u”.

DATA createParamData(String paramNames,TABLE table)

Description: It will create a new table, whose which instances are from “table”, and outputs are “paramNames”. It is used to calculate the parameter values with data.

For example:
table:
W L
1u 1u
1u 10u

model:
…
a = w * l
b = w / l
…

model.createParamData("a,b",table):
W L a b
1u 1u 1e-12 1
1u 10u 10e-12 0.1
