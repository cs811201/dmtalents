Param APIs

double getValue()

Description: Gets the parameter value.

boolean setValue(double value)

Description: Sets the parameter value. If the value is out of the soft boundary, the soft boundary will be extended automatically. If the value is out of the hard boundary, it will be set to the hard boundary.

double getUpper()

Description: Gets the soft upper bound.

double getLower()

Description: Gets the soft lower bound.

boolean setBoundary(double lower, double upper)

Description: Sets the parameter’s soft boundary.

double getHardUpper()

Description: Gets the hard upper bound.

double getHardLower()

Description: Gets the hard lower bound.

boolean setHardBoundary(double lower, double upper)

Description: Sets the parameter’s hard boundary.

double getStep()

Description: Sets the tuning step of parameter.

void extendBound(double ratio)

Description: Extends soft boundary with “ratio” value.

void extendBound(double lowerRatio, double upperRatio)

Description: Extends soft boundary with “lowerRatio” and “upperRatio” values.

boolean reachLowerBound()

Description: Returns true if the parameter value equals to its soft lower bound.

boolean reachUpperBound()

Description: Returns true if the parameter value equals to its soft upper bound.

String getName()

Description: Gets the parameter name.
