PlotCurve APIs

double[] getValues(String expression)

Description: Returns the value of expression.
For example:
double[] v = curve.getValues("ids*vgs");
v[i] is the value of “ids*vgs” of point i
Example 
def plots = cmd.plot("ids_vgs_vbs");
double[] v = plots[0].getCurve(0).getValues("ids");
System.out.println(v[0]);

int getSize()

Description: Returns the number of points.

double getValue(String expression)

Description: Returns the value of an expression.
