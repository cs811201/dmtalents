ScriptDialog APIs

ScriptDialog is used to make some actions easier in Script.
For file chooser, it will record the recent paths.

File selectFile()

Description: Shows a file chooser and returns the file which has been chosen.
Example 
File f = ScriptDialog.selectFile();

File[] selectFiles()

Description: Shows a file chooser and returns the files which have been chosen.
Example 
File[] files = ScriptDialog.selectFile();

File[] selectDir()

Description: Shows a file chooser and returns the directory which been chosen.
Example 
File f = ScriptDialog.selectDir();

File[] selectDirs()

Description: Shows a file chooser and returns the directories which been chosen.
Example 
File[] files = ScriptDialog.selectDirs();

File saveFile()

Description: Shows a file chooser and returns the file which user wants to save.
Example 
File f = ScriptDialog.saveFile();

String getInputString()

Description: Shows an input dialog and returns the String which the user inputs.
Example 
String str = ScriptDialog.getInputString();
