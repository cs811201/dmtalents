POINT APIs

POINT is a special format of DATA, generally a POINT can be regarded as one row of a DATA table. All DATA API can be used in POINT.
Here list all other functions that POINT has besides what it inherits from DATA.

double[] getPropertyValues()

Description: Gets the property values.

double[] getInstValues()

Description: Gets the instance values.

double[] getBiasValues()

Description: Gets the bias values.

double[] getValues(String[] names)

Description: Gets the values with "names".
