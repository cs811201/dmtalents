MBPOPT.POPULATE_PARAM APIs

MBPOPT.POPULATE_PARAM provides built-in methods to deal with multiple results of parameters.

double[][] getValues(Param[] params)

Description: Returns values of params.

double[][] getValues(Param[] params,Param[] reset,sfunc check)

Description: Returns values of params. The "reset" parameter will be generated automatically. And it will make the generated values can access "check".

void updateValue(Param[] params, double[][] values)

Description: Updates values to MBPOPT.POPULATE_PARAM.

void load(File file)

Description: Loads the values from a file.

void save(File file)

Description: Saves the values to a file.

void show()

Description: Shows the values.

void clear()

Description: Clears the values.
