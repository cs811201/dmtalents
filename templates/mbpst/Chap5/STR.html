STR APIs

STR is used to create String more clearly.

If there are many variables in the String, there will be lots of quotation marks, and it could be error-prone. STR can provide convenience for string concatenation.

String STR(String str)

Description: Returns the value of str. If there is a variable, add “#” ahead of it.
Example 
double vgg = 1;
double vdlin = 1;
DATA d = MBPDATA.select("ids_vgs_vbs");
//Normal usage:
d = d.build("select(vgs=="+vgg+"&&vds=="+vdlin+")");

//STR usage:
d = d.build("select(vgs==#vgg&&vds==#vdlin) ");
