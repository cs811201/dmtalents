MBPDATA APIs

MBPDATA provides built-in methods to deal with DATA.
MBPDATA is a system variable but not a class.

DATA get(String name)

Description: Access data from MBP.
MBPDATA.get(“analysis name”);//get dc data
MBPDATA.get(“[device]”);//get dc device
MBPDATA.get(“[rf]y”);//get default y parameter
MBPDATA.get(“[rf]analysis name”); //get y parameter with other rf analysis
MBPDATA.get(“[rfdc]analysis name”);//get dc
MBPDATA.get(“[dp]target name”);//get dp data
MBPDATA.get(“[noise]analysis name”);//get noise data
MBPDATA.get(“[noise][analysis name]op name”);//get noise op
MBPDATA.get(“[stat]target name”);//get statistical data
MBPDATA.get(“[mis]target name”);//get mismatch data
Example 
DATA d = MBPDATA.get("ids_vgs_vbs");
Debug.showData(d);

DATA d = MBPDATA.get("[device]");
Debug.showData(d);

DATA d = MBPDATA.get("[rf]y");
Debug.showData(d);

DATA d = MBPDATA.get("[dp]vth");
Debug.showData(d);

DATA d = MBPDATA.get("[noise]sid");
Debug.showData(d);

TABLE createTable(String instNames,String biasNames,String outputNames)

Description: Creates an empty table.
Example 
DATA d = MBPDATA.createTable("w,l,t","vgs,vbs,vds","ids");
Debug.showData(d);

TABLE createTable(String propertyNames,String instNames,String biasNames,String outputNames)

Description: Creates an empty table.
Example 
DATA d = MBPDATA.createTable("vgg,vdd,vbb","w,l,t","","vth");
Debug.showData(d);

TABLE createTable(String analysisName,String polar,String instNames,String biasNames,String outputNames)

Description: Creates an empty table and this table can be simulated.
Example 
DATA d = MBPDATA.createTable("ids_vgs_vbs","nmos","w,l,t","vbs,vds,vgs","ids");
Debug.showData(d);

DATA merge(DATA[] data)

Description: Merges data array to a larger table.
Example 
DATA d = MBPDATA.get("ids_vgs_vbs");
d = d.build("splitBy(device)").get(0,3);
DATA[] sub = d.toSubArray();
d = MBPDATA.merge(sub);
Debug.showData(d);
