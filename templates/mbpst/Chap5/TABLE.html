TABLE APIs

TABLE is a special format of DATA. Therefore, all DATA API can be used in TABLE.
Here list all other functions that TABLE has besides what it inherits from DATA.

POINT insert(double[] value)

Description: Inserts one line in a table.
Example 
TABLE d = MBPDATA.createTable("w,l","vgs","ids");
d.insert(new double[]{1e-6,2e-6,3,4e-6});
Debug.showData(d);
