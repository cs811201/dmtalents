GraphDataSource APIs

GraphDataSource contains information regarding a graph, such as X/Y/P axis names, graph title, etc.

String getXName()

Description: Gets X axis name of the graph.
Example 
String xName = graphDataSource.getXName();

String getYName()

Description: Gets Y axis name of the graph.
Example 
String yName = graphDataSource.getYName();

String getPName()

Description: Gets P axis name of the graph; P axis in MBP is also known as Z axis, the legend.
Example 
String pName = graphDataSource.getPName();

String getGraphName()

Description: Gets name of the graph.
Example 
String graphName = graphDataSource.getGraphName();

String getDeviceProperty(String propertyName)

Description: Gets device property value of the graph.
Example 
Double w_val = graphDataSource.getDeviceProperty("w");

String getPropertyValue(int curveIndex, String name)

Description: Gets property value of the curve.
Example 
String pName = graphDataSource.getPName();
double pValue = graphDataSource.getPropertyValue(1,pName);
