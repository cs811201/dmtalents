MBPVAR APIs

MBPVAR is used to define global variables, that can be set/get across DATAs, such as IMV Targets, or any user defined DATA, Flow, Setup, etc.

void set(String name, String value)

Description: Sets the value of “name” variable as String “value”.
Example 
MBPVAR.set("var1","value1");
//set the value of var1 as value1

String getString(String name)

Description: Gets the String value of “name” variable.
Example 
String var1 = MBPVAR.getString("var1”);
//get the String value of var1

void set(String name, int value)

Description: Sets the int value to the variable.
Example 
MBPVAR.set("var1",10);
//sets int value 10 to var1

int getInt(String name)

Description: Gets the int value of “name” variable.
Example 
int var1 = MBPVAR.getInt("var1”);
//gets the int value of var1

void set(String name, double value)

Description: Sets the double value to the variable.
Example 
MBPVAR.set("var1",10.1);
//sets double value 10.1 to var1

int getDouble(String name)

Description: Gets the double value of “name” variable.
Example 
double var1 = MBPVAR.getDouble("var1”);
//gets the double value of var1

void set(String name, int[] value)

Description: Sets the int array to the variable.
Example 
MBPVAR.set("var1",new int[]{1,2,3});
//sets int array[1,2,3] to var1

int getIntArray(String name)

Description: Gets the int array value of “name” variable.
Example 
int[] var1 = MBPVAR.getIntArray("var1”);
//gets the int array value of var1

void set(String name, double[] value)

Description: Sets the double array to the variable.
Example 
MBPVAR.set("var1",new double[]{1.1,2.2,3});
//sets int array[1.1,2.2,3.3] to var1

int getDoubleArray(String name)

Description: Gets the double array of the variable.
Example 
double[] var1 = MBPVAR.getDoubleArray("var1");

void show()

Description: Shows the GUI of “VarTable”.
Example 
double[] var1 = MBPVAR.show();
