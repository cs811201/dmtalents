MBP APIs

MBP provides built-in methods to access models and model parameters in MBP. There could be multiple models loaded in MBP at the same time; however, there is only one model called the "Main Model" at a given time, the model that's selected in the model panel currently.

DATA m()

Description: Gets the simulation data.
Example 
DATA d = MBPDATA.get("ids_vgs_vbs");
d = d.s();
