MBP APIs

MBP provides built-in methods to access models and model parameters in MBP. There could be multiple models loaded in MBP at the same time; however, there is only one model called the "Main Model" at a given time, the model that's selected in the model panel currently.

Model getModels().getMainModel()

Description: Returns currently selected model.

int getModels().getModelCount()

Description: MBP may load several models. It returns the number of models loaded in MBP.

Model getModels().getModel(int index)

Description: MBP may load several models. It returns the model with given index.

boolean toBinningMode()

Description: Switches to binning point mode.

DATA getLocalModelManager().createTable(String outputNames)

Description: Creates a table with models loaded in "Local Model Manager"(One can open it with menu "Model->Local Model Manager" in RF Mode).
The instances are defined in "Local Model Manager".
The outputs of measurement are loaded from models in "Local Model Manager".
The outputs of simulation will be calculated with current model.

String[] getDataNames(String type)

Description: Gets the data names loaded in MBP.
Here "type" can be:
"[DC]"— All the DC data names
"[RF]" — All the RF data names
"[DP]" — All the DP data names
"[NOISE]" — All the noise data names
"[STAT]" — All the statistical data names
"[MIS]" — All the mismatch data names"

Param getParam(String name)

Description: Gets the parameters by name.
Example 
Param param = MBP.getParam("u0");

Param[] getParams(String names)

Description: Gets the parameters by names.
Example 
Param[] params = MBP.getParams("u0,k1,vth0");

String getSpiceName()

Description: Gets the SPICE name of the model.

Param createBinningPointDeltaParam(String[] sectionNames,String paramName)

Description: Creates a parameter for binning tweaking.

Param createBinningPointRatioParam(String[] sectionNames,String paramName)

Description: Creates a parameter for binning tweaking.

String[] getBinPointNames(String expression)

Description: Gets the bin names with expression.
For example:
Model model = …;
model.getBinNames("w>1u && l < 1u");
It will return all the bins which satisfy “w>1u && l < 1u”.

DATA createParamData(String paramNames,TABLE table)

Description: It will create a new table, which instances are from “table”, and outputs are “paramNames”. It is used to calculate the parameter values with data.

String getStatParamNames()

Description: Gets the names of statistical parameters.
