MQA Application

Comparison between different versions of the same Simulator

In this video, we will see how to setup MQA to compare models using the same simulator but with different versions.

This application has become practical, number one, since we have accumulated so many simulator versions as time goes by, we have to make sure the consistency; number two, what if our designers or customers are using different versions, it’s not likely we would twist one’s arm to use certain version.

So, MQA is here to bring us the peace of mind, and actually it can mix any simulator of any versions.

In particular, this demo uses ADS 2014 and ADS 2015.
