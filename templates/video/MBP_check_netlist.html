MBP FAQ

Check MBP simulation netlist files

Say we have model and data loaded, and by default MBP uses its internal simulator and netlisting is done in memory so no netlist files are created.

To turn on saving netlist file, please follow the steps in this demo:



    1. switch to external Hspice, or Spectre from the Simulation menu, here we use Hspice as the example.
    2. go to the External Spice options, make sure the checkbox is not checked, indicating I want my netlist files saved.
    3. then click the simulate button from the toolbar to let MBP call hspice.
