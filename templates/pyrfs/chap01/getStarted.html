Get Started

To get started, we need to be able to run the Python code that integrates PyRFS.

As explained HERE, one needs to add $MQAHOME/kedrfs/python directory to PYTHONPATH, as shown below:


Once we can import pyrfs module successfully in our Python environment, we can go ahead to exercise all the examples listed in next Chapter.

Personally, I’d like to use PyCharm, the JetBrains™ PyCharm Community Edition is free to use, and provides all kinds of powerful assistance to help us write Python code.

To enable PyRFS in PyCharm, make sure we do the following:

1. Add the “python.exe” from $MQAHOME/tools/python, which is 2.7.10 version, into the Project’s Interpreter list.



2. Click on the gear icon on the far right, and click on "More...", as shown below.


3. Highlight the one 2.7.10 from MQA, and click the last icon on the vertical bar to the right, it allows us to manage the Interpreter Paths.


4. Add the $MQAHOME/kefrfs/python directory to the Interpreter Paths list.



Now, we shall be able to import “pyrfs” and Run/Debug our Python code, as shown below.



Quickly, check out those amazing examples in Chapter 2!
